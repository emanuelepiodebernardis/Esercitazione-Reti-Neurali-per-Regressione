Questo repository contiene il materiale relativo all’esercitazione obbligatoria
propedeutica all’ammissione all’esame scritto.

L’obiettivo del progetto è analizzare e confrontare le prestazioni di reti neurali
feedforward a singolo strato nascosto su due diversi dataset di regressione,
variando il numero di neuroni e valutando l’errore quadratico medio (MSE)
sia sul training set sia sul test set.
